题目: 给定一个整数数组nums,找到一个具有最大和的连续子数组(子数组最少包含一个元素),返回其最大和;
    输入: [-2,1,-3,4,-1,2,1,-5,4]
    输出: 6
    解释: 连续子数组 [4,-1,2,1] 的和最大,为 6;
1.动态规划:
    [1]思路:
        (1)f(i)代表以第i个数结尾的连续子数组的最大和,问题转换为求max{f(i)}
        (2)要获得f(i)的最大值,取决于 ai和 f(i?1)+ai,我们希望获得一个比较大的;
        (3)由上得出动态规划转移方程: f(i)=max{f(i?1)+ai ,ai}
    [2]实现:
        class Solution {
            public int maxSubArray(int[] nums) {
                // 定义用来保存f(i?1)值的变量
                int pre = 0;
                // 定义用来保存连续子数组最大和的变量
                int maxAns = nums[0];
                // 遍历数组,获取到max{f(i)}并记录
                for(int i : nums) {
                    // 这就是动态规划的转移方程
                    pre = Math.max(pre + i, i);
                    // 这就是连续子数组最大和
                    maxAns = Math.max(maxAns, pre);
                }
                return maxAns;
            }
        }
    [3]复杂度分析:
        (1)时间复杂度:O(n)
            其中n为nums数组的长度,只需要遍历一遍数组即可求得答案;
        (2)空间复杂度: O(1)
            只需要常数空间存放若干变量;
